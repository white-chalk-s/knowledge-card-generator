适用于：资讯门户 / 企业官网 / 行业平台 / 内容首页 / 产品入口页。点击文字可直接改。

虚线 = 可替换图片/插画 绿色 = 主品牌色 灰线 = 组件边界 文字条 = 标题/摘要占位

字体建议

  • 中文：PingFang SC / 微软雅黑
  • 英文数字：Inter / Arial
  • 主标题：28–36px / 800
  • 导航：14px / 700

配色建议

  • 主色：#14A86B，可替换品牌色
  • 背景：#F6F8F7
  • 卡片：#FFFFFF
  • 描边：#D8E0DC

Icon 建议

  • 导航：chevron-down
  • 搜索：search
  • 用户：user
  • 卡片：image / file-text / chart

替换原则

  • 先换行业词，再换图片
  • 不改布局比例
  • 主视觉只放一个核心内容
  • 右侧最多 3–4 条推荐
